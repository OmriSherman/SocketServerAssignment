1. להיכנס לתיקיית code וללחוץ מקש ימני+שיפט על איזור ריק בתיקייה
2. ללחוץ על  open in terminal
3. בטרמינל שנפתח לכתוב "python server.py" 
בעצם כך יפתח הסרבר.
4. שוב מקש ימני באיזור ריק באותה תיקייה ולחיצה על open in terminal
5. בטרמינל שנפתח לכתוב "python client.py"
בחלוֹן הגרפי שנפתח מקלידים שם משתמש, בשדה "Send to" מזינים את שם הנמען ומקלידים את ההודעה בחלק התחתון. לחיצה על Send (או Enter) תשלח את ההודעה.
לחזור על סעיף 5 עבור כל משתמש חדש שנרצה לפתוח ולהתחבר איתו לשרת.
